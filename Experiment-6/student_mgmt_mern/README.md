This repository contains a full MERN-stack Student Management System.
- Backend: Express + Mongoose (REST API)
- Frontend: React (Vite) with Axios and React Router

## Quick start
1. Unzip and open two terminals.
2. Start backend:
   ```
   cd backend
   cp .env.example .env
   npm install
   npm run dev
   ```
3. Start frontend:
   ```
   cd frontend
   npm install
   npm run dev
   ```
4. Open the React app (usually at http://localhost:5173) and the backend runs at http://localhost:5000
