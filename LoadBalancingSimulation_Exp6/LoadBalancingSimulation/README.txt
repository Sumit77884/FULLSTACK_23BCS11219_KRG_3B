HOW TO RUN:
-----------
Option 1 - Command Line (requires Java installed):
  1. Open terminal in this folder
  2. Compile:  javac Main.java
  3. Run:      java Main
  4. Press ENTER when prompted

Option 2 - Online Compiler:
  1. Go to https://www.onlinegdb.com/online_java_compiler
     OR https://www.jdoodle.com/online-java-compiler
  2. Paste the contents of Main.java
  3. Click Run
  4. Press ENTER when prompted in the console
