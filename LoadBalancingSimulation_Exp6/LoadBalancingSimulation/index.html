Key Observations
▶ Horizontal scaling (Exp 2) gave 2× more throughput and 3× faster response vs single server.
▶ Round Robin distributes load evenly — no single server gets overloaded.
▶ Even after Server-2 crashed in Exp 4, system recovered by adding Server-4 automatically.
▶ Fault tolerance caused a temporary dip in throughput (125 → 85 req/s) before recovery.
▶ Stateless servers allow easy horizontal scaling — no session data tied to one machine.
